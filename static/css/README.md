# 🎓 Placement Preparation Assistant

A Flask-based web application designed to help students prepare for campus placements.

## ✨ Features

- 📊 Aptitude Practice
- 💻 Coding Practice (Python, Java, C, C++, SQL)
- 🎤 HR & Technical Interview Preparation
- 📄 Resume Builder
- 🏢 Company Preparation
- 📅 Study Planner
- 📈 Progress Tracker
- 🤖 AI Assistant

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite

## 🚀 How to Run

1. Install Python
2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python app.py
```

4. Open in browser

```
http://127.0.0.1:5000
```
